Lab 2: Simple Data Path

----------
DESCRIPTION

In this lab, the user enters in a number using the keypad, and can save it to any of 4 registers.
The user can also perform addition between 2 registers, and save the sum in a different register.

----------
FILES

-
 Lab2.lgi

 This file is the Multimedia Logic file that the program is contained within

----------
INSTRUCTIONS

This program is meant to be simulated through Multimedia Logic.
Click play and the clear registers button to start.
